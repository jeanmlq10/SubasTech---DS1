# SubasTech Backend

API backend de SubasTech construida con Django y Django REST Framework.

Esta carpeta queda preparada para implementar la API REST, integraciones con WhatsApp, comunicacion con Ollama y persistencia en PostgreSQL.

## Modulos

- `users`: gestion de usuarios y roles.
- `technicians`: tecnicos, especialidades, zonas y disponibilidad.
- `appointments`: citas, estados, cancelacion y reagendamiento.
- `chatbot`: gestion del flujo conversacional.
- `whatsapp`: webhooks e integracion con Meta WhatsApp Cloud API.
- `llm`: integracion con Ollama y Llama 3.1 8B.
