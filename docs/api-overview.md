# API Overview

Estos endpoints representan la superficie esperada de la API. Todavia no estan implementados.

## Endpoints Esperados

### `/api/users/`

Gestion de usuarios, roles y datos basicos de cuentas internas o clientes.

### `/api/technicians/`

Gestion de tecnicos, especialidades, zonas de cobertura y disponibilidad.

### `/api/appointments/`

Gestion de citas, estados, consulta, cancelacion y reagendamiento.

### `/api/chatbot/message/`

Entrada generica para procesar mensajes conversacionales desde canales externos o pruebas internas.

### `/api/whatsapp/webhook/`

Webhook para verificacion y recepcion de eventos de Meta WhatsApp Cloud API.

### `/api/llm/interpret/`

Endpoint interno o de soporte para interpretar mensajes mediante Ollama y Llama 3.1 8B.
