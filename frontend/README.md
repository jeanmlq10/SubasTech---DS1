# SubasTech Frontend

Aplicacion frontend de SubasTech construida con Next.js.

Esta carpeta queda preparada para implementar posteriormente el panel administrativo, vistas de seguimiento y herramientas de soporte.

## Estructura

- `app/`: rutas y layouts de Next.js.
- `components/`: componentes reutilizables.
- `lib/`: utilidades compartidas.
- `services/`: clientes para consumir la API del backend.
- `types/`: tipos compartidos del frontend.
